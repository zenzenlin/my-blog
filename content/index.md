# 歡迎來到我的部落格

這是一個使用 Nuxt Content 建立的部落格文章。

## 為什麼選擇 Nuxt Content？

Nuxt Content 提供了以下優點：

- 使用 Markdown 撰寫文章
- 支援程式碼高亮
- 自動生成目錄
- 支援 YAML front matter
- 強大的查詢 API

## 開始使用

你可以在 `content` 目錄下創建更多的 `.md` 文件來添加新的文章。
